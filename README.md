# 📚 Online Book Borrowing

**Online Book Borrowing** is an online book borrowing platform where users can explore books, check availability, and borrow them. The goal is to provide a simple and user-friendly system for managing book borrowing digitally.

## 🌐 Live URL
https://my-ph-a8-online-book-borrowing-plat.vercel.app/

## ✨ Key Features
- 📖 Browse books by category (Story, Tech, Science)
- 🔍 Search and view book details
- 📚 Borrow and return books
- 👤 User authentication (Google / Email login)
- 🎨 Responsive UI using modern component libraries

## 📦 NPM Packages Used
### Category	Packages
- 🚀 Core	next, react, react-dom
- 🎨 UI & Styling	tailwindcss, daisyui, @heroui/react, @gravity-ui/icons, react-icons
- 🔐 Authentication & Database	better-auth, @better-auth/mongo-adapter, mongodb
- 🧩 Form & UX	react-hook-form, react-toastify, react-fast-marquee
- 🛠️ Development Tools	eslint, eslint-config-next


## ⚙️ Installation
- git clone https://github.com/mdferdowsanwar/my-ph-a8-online-book-borrowing-platform
- cd your-repo
